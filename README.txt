or NSMWHF for short

Made by Souldbminer, YoshiFanatic1 and DannyLeeClark
INTRODUCTION:
Nsmwhf is a framework for Super Mario World rom hacking made by Souldbminer and Yoshifanatic1 based upon SMWDisX

This program requires an installation of Python 3 to function properly. Faliure to install Python 3 will prevent builds from working properly

FEATURES:
C++-like configuration files
Source can be compiled from C automatically
Full support for Git version control
Supports Windows, MacOS and Linux
Advanced editor (Volcano Lotus) based on LDtk
Easy Integration of many popular modifications
Conversion scripts to allow easily porting existing modifications
Automatic graphics ripping and editing from .ase files
Up to 256MB of rom space
.wav and .mp3 audio support
